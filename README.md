Site institucional desenvolvido para a Clínica fictícia Naju, com foco em apresentação profissional, navegação fluida e experiência moderna para o usuário.

--

# Sobre o Projeto

A proposta do site é transmitir:

- Confiança
- Sofisticação
- Clareza nas informações
- Facilidade de navegação

O layout foi construído com seções bem definidas, navegação com destaque dinâmico da seção ativa e header fixo com comportamento responsivo ao scroll.

--

# Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- IntersectionObserver API
